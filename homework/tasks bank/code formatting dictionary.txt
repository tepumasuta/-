1. Тэги принято писать в нижнем регистре
2. Объявление типа документа принято писать в верхнем регистре
3. При присваивании значения атрибуту принято использовать двойные кавычки
4. Тэги html, body и head принято писать на уровне одного отступа
5. Между тэгами принято писать информацию без пробелов
6. Тэги, содержащие контент, помещающийся в одну строчку, следует писать на одной строчке (<тэг>контент...</тэг>)
